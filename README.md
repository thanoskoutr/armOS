# minimARM-OS

## Boards
Support for:

- [x] Raspberry Pi Zero W
- [x] Raspberry Pi 4
## Features
- [x] Basic Directory Structure
- [x] Register Addresses
    - [x] GPIO
    - [x] AUX
- [x] mini UART
- [x] Kernel printing info on start up
- [x] `printk`
- [x] `string.h` functions
    - [x] `strlen`, `strcmp`, `strcpy`, `strcat`, `strrev`
    - [x] `memset`, `memcpy`
- [ ] `stdlib.h` functions
    - [x] `itoa`
- [x] `stdbool`
    - [x] `true`, `false`
