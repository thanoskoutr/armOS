# minimARM-OS

## Boards
Support for:

- [x] Raspberry Pi Zero W
- [x] Raspberry Pi 4
## Features
- [x] Basic Directory Structure
- [x] Register Addresses
    - [x] GPIO
    - [x] AUX
- [x] mini UART
- [x] Kernel printing info on start up
- [x] `printk`
- [ ] `string.h` functions
    - [x] `strlen`, `strrev`
    - [ ] `strcat`, `strcmp`, `strcpy`
    - [ ] `memset`, `memcopy`
- [ ] `stdlib.h` functions
    - [x] `itoa`
